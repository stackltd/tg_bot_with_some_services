### Руководство по эксплуатации телеграм-бота

#### Данный бот предосталвяет следующие сервисы:
1. Поиск по гео-координатам **_/geo_**
2. Википедия **_/wiki_**
3. Оффлайн плей-лист Youtube **_/youtube_**
4. Информация по IP адресу **_about_IP_**

### Настройка
В .env необходимо указать параметры:
```
TOKEN=ваш_тг_токеn
my_uid=ваш_uid_в_tg
```
